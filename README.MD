# Scorable Examples

This repository contains example applications demonstrating the usage of Scorable system.

## 🧑‍🍳 Examples

- [Poker App](./poker-app) - A simple poker CLI application showcasing how to create and use a custom evaluator.
- [SDK Examples](https://github.com/scorable/rs-python-sdk/tree/main/examples) - Code snippets using the Scorable Python SDK.
- [RAG Hallucination Detection](https://colab.research.google.com/drive/1A1mmE2jHVntSJiXzBIpMpDjxrtgWC8uq?usp=sharing) - A Colab notebook showcasing how to detect RAG hallucination using Scorable.

## 📖 Getting Started

Each example application contains its own README with specific setup and running instructions.
